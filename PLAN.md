# Kế hoạch tạo trang web nhà hàng Châu Âu cổ điển

## 1. Thông tin thu thập được

- Yêu cầu: Tạo trang web nhà hàng theo phong cách châu Âu cổ điển
- Loại: Trang chủ (homepage)
- Phong cách thiết kế: Cổ điển, sang trọng, vintage châu Âu

## 2. Các trang cần tạo

- `index.html` - Trang chủ chính
- `style.css` - File CSS cho style
- `script.js` - File JavaScript cho tương tác

## 3. Cấu trúc trang chủ

- **Header**: Logo nhà hàng, menu điều hướng
- **Hero Section**: Hình ảnh background sang trọng, tên nhà hàng, slogan
- **About Section**: Giới thiệu về nhà hàng, lịch sử
- **Menu Highlights**: Một số món ăn đặc trưng
- **Reservation CTA**: Kêu gọi đặt bàn
- **Footer**: Thông tin liên hệ, địa chỉ, mạng xã hội

## 4. Phong cách thiết kế

- Màu sắc: Vàng đồng, nâu sẫm, vàng nhạt, đen
- Font chữ: Serif cổ điển (Playfair Display, Cormorant Garamond)
- Hoa văn trang trí: Victorian/ornamental
- Hiệu ứng: Fade-in, parallax

## 5. Các bước thực hiện

- [] Tạo file index.html với cấu trúc trang
- [] Tạo file style.css với phong cách châu Âu cổ điển
- [] Tạo file script.js cho hiệu ứng tương tác
- [] Kiểm tra và xác nhận
